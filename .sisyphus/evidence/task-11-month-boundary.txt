## Month Window Implementation

The month window query in internal/http/http.go:

```go
func (s *Server) queryMonth(...) {
    now := time.Now().UTC()
    startOfMonth := time.Date(now.Year(), now.Month(), 1, 0, 0, 0, 0, time.UTC)
    end := storage.CurrentDateUTC()
    start := storage.FormatDateUTC(startOfMonth)
    return s.queryDateRange(start, end, ...)
}
```

- start = first day of current calendar month at 00:00:00 UTC
- end = today (current UTC date)
- Query uses: ListDaily(ctx, start, end, interfaceFilter) with date_utc >= start AND date_utc <= end

## UTC Boundary Semantics

- time.Now().UTC() provides current moment in UTC
- time.Date(year, month, 1, 0, 0, 0, 0, time.UTC) creates start-of-month at midnight UTC
- FormatDateUTC converts to YYYY-MM-DD string for SQL comparison
- CurrentDateUTC returns today's date as YYYY-MM-DD in UTC
- SQL comparison is string-based lexicographic on YYYY-MM-DD format, which matches chronological order

## Flush Loop Rollover Behavior

In internal/runtime/flush.go Flush():
- currentDateUTC := storage.CurrentDateUTC() (called once at start of flush)
- Checkpoint date comparison: cp.DateUTC == currentDateUTC
- If same day: compute delta from checkpoint
- If new day: use raw counters as delta (counter wrap handled)
- persistAll writes checkpoint with currentDateUTC, upserts to totals, upserts to daily bucket

This ensures:
1. New UTC day = new daily bucket automatically
2. No cross-day accumulation in same bucket
3. Checkpoint date tracks which day the counters are from
