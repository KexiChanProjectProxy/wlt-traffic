=== SQL ON CONFLICT Behavior ===

All upserts use SQLite ON CONFLICT DO UPDATE SET ... = ... + excluded.* pattern:
- totals: ON CONFLICT(interface_name, ifindex, mac) DO UPDATE SET bytes = bytes + excluded.bytes, ...
- daily: ON CONFLICT(interface_name, ifindex, mac, date_utc) DO UPDATE SET bytes = bytes + excluded.bytes, ...
- checkpoints: ON CONFLICT(interface_name, ifindex, mac) DO UPDATE SET last_raw_bytes = excluded.last_raw_bytes, ...

=== Transactional Safety ===

All upserts use explicit transaction pattern:
1. tx.BeginTx(ctx, nil)
2. tx.ExecContext(ctx, query, ...)
3. tx.Commit()
4. tx.Rollback() on error path

- No duplicate rows created
- Updated_at timestamp updated on each modification
