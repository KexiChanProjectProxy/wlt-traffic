Shutdown Cleanup Implementation:
-------------------------------

Service.Stop() method performs the following cleanup:

1. Calls manager.DetachAll() which iterates all tracked interfaces
2. For each interface, detaches both ingress and egress directions
3. Updates state to StateDetached after successful close
4. Collects any detach errors for reporting
5. Marks all interfaces as detached in status
6. Sets service mode to ModeFailed to indicate shutdown

Cleanup Flow:
------------
Service.Stop()
  -> manager.DetachAll()
     -> for each ifaceName in ifaceStates:
        -> DetachIface(ifaceName)
           -> detachDirection(ifaceName, DirectionIngress)
           -> detachDirection(ifaceName, DirectionEgress)
              -> if StateAttached and Link != nil:
                 -> Link.Close() 
                 -> set StateDetached
  -> mark all interfaces detached in status
  -> print any detach errors

Idempotency:
-----------
- detachDirection checks state before attempting close
- If state is not StateAttached or Link is nil, returns StateDetached immediately
- Safe to call Stop() multiple times

Detach State Transitions:
------------------------
StateAttached -> StateDetached (successful close)
StateAttached -> StateFailed (close error)
Any -> StateDetached (Stop called)

Link Types:
----------
- Real TCX links: link.Link from AttachTCX() - properly closes kernel hook
- Mock links: nil in mock mode - no-op cleanup

Error Handling:
--------------
- Detach errors collected and printed but don't stop cleanup
- Individual detach failures don't prevent other interface cleanup
- Service continues shutdown even with partial detach failures

Runtime Integration:
------------------
Service.Stop() is called from main.go signal handler:
  sigCh := make(chan os.Signal, 1)
  signal.Notify(sigCh, syscall.SIGINT, syscall.SIGTERM)
  select { case sig := <-sigCh: cancel() }
  -> svc.Stop() called
